1) To check a computers network configuration:

Command: ipconfig

This command helps us check if the IP address of the machine is a valid IP address or not.

Information to look out for: Host name, IP address and Network Mask.

2) Command: ipconfig/all

This will give you the internet configuration of the computer system and the node, tunnel, mask related information

3)command: ipconfig/all

Under the information displayed, physical address and IPv4 address respectively show the MAC address and Network configuration.
